  1. What’s a superbaud? Multi-h continuous phase modulations use a set of distinct modulation indices (one index per symbol) in a periodic manner, and the term superbaud refers to the period of repetition of modulation indices (and superbaud timing refers to knowing which symbol corresponds to which modulation index).
